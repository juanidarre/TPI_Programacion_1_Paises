Descripción del Proyecto
Este sistema es una aplicación de consola desarrollada en Python 3.x para la gestión y análisis de datos demográficos y geográficos de diversos países. El programa procesa información proveniente de un archivo estructurado en formato CSV y permite realizar operaciones avanzadas de búsqueda, filtrado y cálculos estadísticos.

Funcionalidades principales:
Carga de datos: Lectura robusta desde `paises.csv` con manejo de excepciones.
Búsqueda: Localización de países por nombre (coincidencia parcial o exacta).
Filtrado: Segmentación de datos por continente.
Ordenamiento: Implementación manual del algoritmo para ordenar por población.
Estadísticas: Cálculo de promedios de población y superficie, identificación de valores máximos/mínimos y conteo de países por continente.

---

Instrucciones de Ejecución
Para correr el programa en la computadora, siga estos pasos:

1.  Requisitos previos: Asegúrese de tener instalado Python 3.x.
2.  Preparación de archivos: El archivo `main.py` y el archivo `paises.csv` deben estar en la misma carpeta.
3.  Ejecución:
     -  Abra una terminal o consola.
     -  Navegue hasta la carpeta del proyecto.
     -  Ejecute el comando: `python Main.py`
